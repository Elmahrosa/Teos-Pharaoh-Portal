# TEOS Egypt — Contributing Guide

## Welcome
Thank you for your interest in contributing to Pharaoh Portal.  
All contributions are subject to **sovereign compliance review**.

## How to Contribute
1. Fork the repository.
2. Create a feature branch (`feature/my-update`).
3. Commit changes with clear messages.
4. Submit a Pull Request (PR).

## Requirements
- Follow repo architecture and branding guidelines.
- Include tests for new features.
- Respect CODEOWNERS and branch protection rules.
- Contributions automatically transfer ownership to TEOS Egypt.
